GarryMPN

GarryMPN is a program to make simple Player Model & NPC Garry's Mod addon structures with an MDL model and VTF textures.

Note that this program mainly automates taking the puzzle pieces of models and textures and making an addon out of those. You won't get super complicated addons made with this program alone.

You can safely edit the output of GarryMPN's addons once made. Feel free to add your own logic or settings to the output files, or maybe make an issue on the GitHub repository requesting for your feature!

VTFEdit is a .NET-based GUI tool for the creation and conversion of VTF and VMT files.
GarryMPN utilizes VTFCmd to convert PNG images to VTF files. VTFLib and it's default GPL applications are included within the program's files, with their original licenses and files attached.
VTFLib is an LGPL open source programming library, created by Neil "Jed" Jedrzejewski and Ryan "Nemesis" Gregg.
VTFCmd is a command-line based tool that allows you to convert standard image files into the Valve Texture Format (.VTF)
